# Investigation Findings

## Network Evidence (Wireshark)
- Suspicious DNS queries to unknown domains
- HTTP GET request indicating a software download
- Download activity observed over HTTP

## Disk Evidence (Autopsy)
- Browser history entries found in the sample disk image
- Download-related artifacts identified
- Demonstrated how browser activity is recorded on disk

## Conclusion
This task demonstrated how network-based evidence and disk-based artifacts
can be analyzed together to understand user activity during a forensic investigation.
